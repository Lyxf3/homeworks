## Нужно обойти объект таким образом, чтобы сначала в консоль писались самые ближайшие узлы
 
```js
log({
  a: {
    l: {u: 8},
    b: 1
  },

  c: {
    d: 2,
    e: {f: 7}
  },

  j: 4
})

// 4
// 1
// 2
// 8
// 7
```

## Вернуть сумму всех цифровых элементов объекта (в том числе вложенных)

```js
sum({
  a: {
    b: 1,
    c: {
      e: 2,
      d: 10
    }
  }
}) === 13
```
